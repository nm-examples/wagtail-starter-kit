Developer Tools

Model Inspector is a tool that lists models by inspecting all the contenttypes present in the wagtail site.
You can use the action buttons to go directly to an admin or frontend page for a related model and check it's response.
When is this useful?
  1. Following the addition of new features.
  2. Check an upgrade hasn't introduced custom template errors.
{% if not settings_present %}

Optional Configuration: The inspector can be customised by adding MODEL_INSPECTOR_EXCLUDE to your sites settings.
There you can add a tuple for each (app_name, model) to exclude it from the listing. See the package documentation for further details.
